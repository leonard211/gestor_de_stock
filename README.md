# 📦 Gestão de Stock

Programa simples em Python para gerir materiais em stock através do terminal.

## 🚀 Funcionalidades
- Adicionar materiais
- Consultar stock
- Atualizar quantidades (entrada/saída)
- Ver estado geral do stock

## ▶️ Como executar
1. Certifica-te que tens Python instalado
2. Corre o programa:

```bash
python stock.py
